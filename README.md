## 📌 Overview

This project benchmarks three AI coding assistants on common programming tasks to evaluate their performance in real-world development scenarios.

**Tools compared:**

* ChatGPT-4o
* GitHub Copilot
* Gemini 1.5

---

## 🎯 Objective

The goal is to analyze each tool based on:

* Accuracy
* Response speed
* Code quality
* Overall efficiency

---

## 🧪 Tasks Used

* Hello World (basic coding)
* FizzBuzz logic
* Simple REST API generation
* Concept explanation
* Unit test generation

---

## 📊 Results Summary

| Tool           | Quality Score | Accuracy | Speed   |
| -------------- | ------------- | -------- | ------- |
| ChatGPT-4o     | 9.1 / 10      | 91%      | 3.4 sec |
| GitHub Copilot | 8.4 / 10      | 84%      | 2.1 sec |
| Gemini 1.5     | 7.8 / 10      | 78%      | 1.8 sec |
